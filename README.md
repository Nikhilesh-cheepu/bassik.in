# Bassik Reservations Hub

A modern, mobile-first booking hub for venue reservations.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- React 18

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```

## Project Structure

```
app/
├── api/reservations/route.ts
├── page.tsx
└── reservations/page.tsx
components/
├── BrandTabs.tsx
├── Navbar.tsx
└── ReservationForm.tsx
lib/
└── brands.ts
```

## API

POST `/api/reservations` - Handles reservation submissions via WhatsApp
